## Output Format: PDF

### Installation

- **Windows**: Install [MikTeX](https://miktex.org).
- **Linux**: Install `TeX Live` on your system. (e.g. for Ubuntu, `sudo apt-get install texlive`)

### Generate Output

```bash
pdflatex -output-directory=output amiv-statuten.tex
```
